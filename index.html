HTML

Structure & Semantics

HTML gives every page its skeleton — headings, paragraphs, images, links, and more. Semantic elements like <header>, <nav>, and <main> make content accessible and meaningful.

<header>
  <h1>Hello, World!</h1>
</header>
<main>
  <p>Welcome to my site.</p>
</main>
CSS

Style & Layout

CSS transforms raw HTML into beautiful interfaces. Flexbox, Grid, custom properties, and transitions give you full creative control.

:root {
  --primary: #6c5ce7;
}

body {
  display: grid;
  place-items: center;
  font-family: "Inter", sans-serif;
}
